# Vote-Project
팀 투표 시스템 - 팀원 중심의 팀 투표 시스템, 관리자가 조장과 조를 생성하면 팀원들이 지원 후 다른 모든 팀원들의 투표를 기다림. Servlet / JSP 및 Mybatis(DAO)로 구현 : 프론트 컨트롤러 패턴으로 일괄처리 적용
